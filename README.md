🎮 Tic Tac Toe in Python
🎯 Purpose
This project implements the classic Tic Tac Toe game in two versions:

Console-based Python version – text-driven gameplay.
<img width="500" height="500" alt="image" src="https://github.com/user-attachments/assets/d9f5dbe4-6032-40ef-aeb2-8832e0ab4760" />
Graphical pygame version – interactive GUI with mouse input.
<img width="500" height="500" alt="image" src="https://github.com/user-attachments/assets/80b4684d-270f-49f1-b5ca-ea8618b99991" />
The project demonstrates how to apply software engineering principles to a small but complete game.

Development Process Chosen: Agile (iterative improvements, testing, and feedback).

Reason for Choice: Agile allows incremental refinement (starting with a console version, then extending to a GUI).

Possible Usage: Educational tool for beginners learning Python, pygame, and basic AI algorithms.

🛠️ Software Development Plan
Members & Roles

Developer A: Core game logic (win conditions, AI).

Developer B: Pygame interface (graphics, event handling).

Developer C: Documentation & testing.

Developer D:

Schedule

Week 1: Console version completed.

Week 2: Pygame version implemented.

Week 3: Testing, debugging, and documentation.

Algorithm

Heuristic AI: Computer checks for winning moves, blocks user moves, prioritizes center/corners, and falls back to random moves.

Current Status

Both versions run successfully.

Future Plan

Implement Minimax AI for unbeatable computer play.

Add score tracking and restart options.
